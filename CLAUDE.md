# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

This project supports building the portfolio of a graphic designer. The work involves collecting the designer's projects, designing how each piece is presented, and writing copy tailored to each project. Later, this content may be moved into a dedicated portfolio website.

At this stage we are only setting up the project skeleton — no application code or framework has been built yet.

## `.claude/` Directory

Project-specific customizations for Claude Code live under `.claude/`:

- `.claude/agents/` — custom agents tailored to this project
- `.claude/skills/` — custom skills
- `.claude/commands/` — custom slash commands

These directories are currently empty and will be filled in as the project grows.

## Notes

The `.obsidian/` directory contains Obsidian app configuration and should not be modified unless working on the vault configuration specifically.
